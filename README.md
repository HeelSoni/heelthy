# Heelthy - 100% Pure Vegetarian

Heelthy is a modern, 3D interactive web application showcasing vegetarian and diet foods. Built with a focus on premium aesthetics and smooth, dynamic design, it features scrollytelling elements and 3D culinary navigation designed to "wow" the user.

## ✨ Features
- **3D Navigation:** Interactive 3D elements for discovering different healthy dishes.
- **Scrollytelling:** Scroll-linked animations and parallax transitions.
- **Premium Design:** Glassmorphism UI, high-end typography, and dynamic hover effects.
- **Fully Responsive:** Sleek experience across desktop, tablet, and mobile displays.

## 🚀 Tech Stack
- Frontend: React 19 + TypeScript
- Build Tool: Vite
- Styling: Tailwind CSS v4 & Framer Motion for animations
- 3D Graphics: Three.js / R3F (React Three Fiber)

## 📦 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed, and ideally [pnpm](https://pnpm.io/) since this project uses a `pnpm-workspace`.

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/HeelSoni/heelthy.git
   cd heelthy
   ```

2. Install dependencies:
   ```sh
   pnpm install
   ```

3. Run the development server:
   ```sh
   pnpm run dev
   ```

4. Build for production:
   ```sh
   pnpm run build
   ```
